This is a simple inventory tarcking application that can perform basic CRUD operations
 and track the inventory as it is added to tempory shipments or when shipments are placed
  or cancelled.

This program uses php and session variables as a proxy database.

Using the counter one can add items up to inventory quantity to a shipment.
Shipments can then be confrmed or cancelled.

Entries can be edited where details can be updated.

Entries can be deleted where a deletion confirmation page while show.
